# IMDb SQL Data Analysis Project

This project explores large IMDb datasets using SQL, focusing on performance optimisation and handling big data efficiently.

## 📊 Project Overview

The dataset contains millions of records, which created challenges in processing speed and query performance.

To solve this, I applied:

* SQL joins
* Window functions (RANK)
* Indexing
* Data splitting techniques

## ⚡ Key Achievement

One of the queries was optimised from an estimated execution time of **11.5 days to just 1–2 seconds**.

## 🛠️ Tools Used

* MySQL
* MySQL Workbench
* Excel (for generating LOAD DATA commands)
* Git (for file splitting tools)

## 📌 Key SQL Techniques

* INNER JOIN
* Subqueries
* Window functions (RANK)
* Indexing
* Query optimisation
